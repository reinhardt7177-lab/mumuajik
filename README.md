# 🥁 무무드럼

> **나만의 비트를 만들어 보세요!**
> mumuclass by 불곰코끼리

---

## 🎮 이게 뭐야?

브라우저에서 바로 실행되는 **웹 기반 드럼머신**이에요.
클릭 몇 번으로 나만의 드럼 비트를 만들고, 키보드로 드럼도 칠 수 있어요.
설치 없이, 파일 하나만 있으면 돼요.

---

## 🚀 실행 방법

```
무무드럼.html  ← 이 파일 하나만 더블클릭!
```

별도 설치, 서버, 인터넷 연결 **전혀 필요 없어요.**
Chrome / Edge / Firefox 최신 버전 권장.

---

## 🎵 주요 기능

### 드럼 시퀀서
- **4트랙** (Kick / Snare / Hi-Hat / Clap) × 최대 **32스텝** 격자
- 셀 클릭으로 비트 ON/OFF
- **8 / 16 / 32 스텝** 자유롭게 전환
- **BPM 60~180** 슬라이더 조절

### 소리
- 외부 파일 없이 **Web Audio API**로 드럼 소리 합성
- 트랙별 **뮤트(M)** 버튼

### 예시 비트 11종
| 장르 | 프리셋 |
|------|--------|
| 기본 장르 | 🤘 록 · 🎤 힙합 · 🌴 레게 · 🎺 삼바 · 🕺 펑크 |
| 유명 비트 | 🤟 Back in Black · 🌊 보사노바 · 🪩 디스코 · 🎷 재즈 · 🎮 드럼앤베이스 · 🔫 트랩 |

### 배경음악
- 🎵 버튼으로 ON/OFF
- 프리셋 선택 시 **장르에 맞는 배경음악 자동 전환**
  - 록 → 디스토션 기타 베이스
  - 힙합 → 묵직한 808 서브베이스
  - 트랩 → 808 피치 슬라이드
  - 재즈 → 워킹 베이스
  - 보사노바 → 나일론 기타 아르페지오
  - 펑크/디스코 → 움직이는 베이스라인
  - 레게 → 오프비트 코드 스텝
  - 드럼앤베이스 → 빠른 신스 아르페지오

### 🎲 랜덤 비트
- 버튼 하나로 음악적 확률에 기반한 비트 자동 생성
- 클릭할 때마다 새로운 비트!

### ⌨️ 키보드 연주
| 키 | 소리 |
|----|------|
| `D` | Kick |
| `F` | Snare |
| `J` | Hi-Hat |
| `K` | Clap |
| `Space` | Play / Stop |

### 패턴 저장
- 내 비트를 **최대 5개** 이름 붙여 저장
- 새로고침해도 유지 (LocalStorage)

### 오디오 비주얼라이저
- 재생 중 실시간 주파수 스펙트럼 표시

---

## 🛠️ 기술 스택

- **HTML + CSS + JavaScript** (프레임워크 없음)
- **Web Audio API** — 드럼 합성 & 배경음악 합성
- **Canvas API** — 실시간 비주얼라이저
- **LocalStorage** — 패턴 저장
